# Service Roster Prototype

A lightweight prototype for planning:

- Adonai (Saturday)
- Sunday Service

It includes:

- Editable assignment cards
- Browser local storage persistence
- Print-friendly PDF export for one service or the whole weekend

## Open it

Open `index.html` in a browser.

## Save as PDF

Use either `Save PDF` button. Your browser print dialog will open, and you can choose `Save as PDF`.
